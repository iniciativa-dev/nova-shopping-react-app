## Modo de usar

### Git Bash (SSH)

     $ git clone git@github.com:iniciativa-dev/nova-shopping-react-app.git

### Terminal (Dentro da pasta)

     $ npm install

### Rodando em um servidor local (http://localhost:3000)

     $ npm start
